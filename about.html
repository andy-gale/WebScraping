This project is focused on web scraping prices of different graphics cards sorted based on the different base models within NVIDIA 30 series and integrating the scraped data into a Google Sheet. It has Python scripts utilizing the BeautifulSoup library for web scraping and the gspread library and google sheets API for integrating google sheet data. We utilize d3.js for that charts to visualize the data in a mixed dot plot and line plot with the raw price data being displayed as well as tracking mean and median values of price data over time. We have a python script to run all of our scrapers together and use github actions to automate this script to be run daily.
